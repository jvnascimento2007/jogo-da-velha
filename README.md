# Jogo da Velha
Feito em Vanilla JS usando bundler Vite e gerenciador de pacotes Bun

## Rode esse código
Baixe esse repositorio
```terminal
git clone https://github.com/jvnascimento2007/jogo-da-velha.git
cd jogo-da-velha
```

Inicie servidor de desenvolvimento
```terminal
bun run dev
```

Faça build do projeto
```terminal
bun run build
```

Veja como roda em produção
```terminal
bun run preview
```
